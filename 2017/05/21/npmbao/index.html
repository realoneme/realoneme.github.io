一些有趣的npm包

用些小工具可以事半功倍哦！

nrm模块

默认下载模块通过的是npm下载，切换到国内(cnpm,taobao)
npm install nrm -g
安装后就有全局命令nrm
nrm test展示每个镜像源的网速
nrm use用哪个源

http-server

  • 帮助用户启动一个服务，返回静态文件
    npm install http-server -g

nodeppt

npm install nodeppt -g

  • 在想要执行的文件夹下，文件中包含slide

yarn 包管理器
